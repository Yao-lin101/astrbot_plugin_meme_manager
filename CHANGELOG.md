# Changelog

## v4.8.1 (2026-06-02)

### ✨ 新增
- **自适应暗色模式**：页面自动适配系统暗色模式 (`prefers-color-scheme: dark`)，覆盖所有组件（卡片、弹窗、侧边栏、标签、搜索框等）
- **并发数可配置**：「批量重新分析」模态框中新增并发数量选择器（1-10），用户可自行控制速度与资源占用
- **孤立标签一键清理**：「其他」下拉菜单新增「清理孤立标签」功能，自动删除无表情包使用的标签配置

### 🔧 修复 & 优化
- **批量分析超时调整**：LLM 请求超时 30s → 120s，适配 GIF 等复杂图片的分析，支持最多 3 次重试（指数退避）
- **取消不再浪费 token**：批量分析取消后，排队中和正在读取文件的协程立即停止，不再发送 LLM 请求
- **CSS 变量补全**：新增 `--bg-secondary`、`--bg-input` 变量，避免暗色模式下的浏览器默认白色回退
- **暗色覆盖全面**：30+ CSS 类 + 内联样式 `[style*="background:..."]` 属性选择器全量覆盖
